About

NeuroSurface is a Three.js-based brain surface visualization library designed for displaying multi-layer fMRI activation data. It supports multiple surface formats including GIFTI, FreeSurfer, and PLY.

The test data includes brain surfaces from the GIFTI-Reader-JS project, which are real neuroimaging surfaces suitable for testing visualization features.
